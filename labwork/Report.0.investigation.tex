\section{How to compile}
First, cd to folder "labwork".

Then, use commands: \\
\begin{verbatim}
>> cmake CMakeLists.txt
>> make -j    
\end{verbatim}


\section{Condition to compile}
- NVIDIA driver 

- CUDA SDK 7+

- A C++ compiler (such as g++ 5.x.x and above)
